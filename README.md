# LE RECYCLE 前端部分

## 简介
这是LE RECYCLE应用的前端部分，使用Vue+VueRouter+Vuex+VueResource进行开发，使用ES6语法进行代码编写，使用webpack+gulp搭建脚手架，使用Less预编译css。

## 基本功能
用户可以注册并登陆应用，支持回收点查询、投放电池以及积分兑换等操作


##使用

```
$ git clone https://github.com/HUJINLIANG/battery_recycle_fe.git
$ cd battery_recycle_fe
$ npm install
```


## dev
```
gulp serve
```
在浏览中打开http://localhost:3000

## prod
 
```
$ npm start
```

## 布署

使用pm2进行部署
